Why I Build These

Epidemiologic Rigor, Applied to AI Systems

Epidemiology trained me to think in workflows and decision logic — eligibility criteria, structured data capture, safety protocols, audit trails. Building AI agents turned out to be a natural extension of that same skill set: defining clear inputs, decision points, and outputs, then automating the workflow end-to-end.

Each of these systems was built solo — architecture, backend, API integrations, and (where relevant) the conversational or decision logic itself. They range from a voice-based clinical intake agent built for a federally funded maternal health program, to a fully autonomous content pipeline, to a LangGraph-based job-search agent that has actively supported my own search for RWE and HEOR roles.

I see this as directly complementary to clinical epidemiology and RWE work — the same discipline that produces valid causal estimates also produces AI systems with embedded safety logic, structured outputs, and reproducible behavior.

AI Agent · Voice Interface · Healthcare Intake

GRACE — Guided Responsive AI Conversation Engine

A prototype voice-enabled AI intake agent built for Healthy Start SC caseworkers — replacing a static paper-based intake form with a conversational agent that conducts the full federal Pregnancy Risk Assessment interview, including clinical safety screening, in natural spoken dialogue.

🎙️ Voice input → Claude API conversation logic → EPDS / IPV safety checks → 🔊 ElevenLabs voice reply → REDCap-ready output
  • Used Claude API as the conversational engine, ElevenLabs TTS (Sarah voice) for natural voice synthesis, and the Web Speech API for live microphone input — creating a full two-way voice interview experience
  • Designed EPDS depression screening and IPV safety-flagging logic directly into the AI system prompt, embedding clinical safety protocols at the model level with staff-only field protections
  • Built JWT-based staff authentication, a participant-facing voice interview interface, and automated document generation producing REDCap-ready data entry sheets from each completed interview
  • Covers the full Healthy Start SC Pregnancy Risk Assessment form — translating a 30+ field clinical intake into a guided, adaptive conversation

AI Agent · Content Automation · Multi-API Orchestration

Automated Marketing Video Pipeline

A fully autonomous content pipeline that researches trending topics, writes its own scripts, generates voiceovers, assembles finished video, and publishes across platforms — running on a daily schedule with no manual editing step.

🔎 Trend detection → Claude API script → ElevenLabs voiceover → Shotstack assembly → AssemblyAI captions → 📺 YouTube / TikTok publish
  • Built an automated pipeline that researches trending topics, generates AI-written scripts via Claude API, synthesizes voiceovers with ElevenLabs, and assembles ~7-minute stock-footage videos with auto-captions using Shotstack and Pexels
  • Integrated seven external APIs — trend detection, LLM scripting, TTS, video rendering, transcription (AssemblyAI), and YouTube publishing — into a single modular pipeline with full logging and error handling
  • Implemented YouTube OAuth2 refresh-token authentication for unattended daily uploads, automated affiliate-link substitution, and a TikTok export module producing platform-optimized 58-second cuts with overlay branding
  • Runs end-to-end on a daily schedule — from topic discovery to multi-platform publishing — with no manual editing step

AI Agent · Job Search Automation · LangGraph

OutreachIQ — Networking Outreach AI Agent

An autonomous agent built to scale and systematize my own job search — finding relevant research, epidemiology, biostatistics, and public health roles, scoring fit against my qualifications, and drafting personalized outreach without manual effort for every application.

🔎 Job discovery → LLM fit scoring → Personalized email/cover letter draft → 📧 Gmail send → PostgreSQL tracking
  • Developed an autonomous agent using LangGraph that identifies relevant research, epidemiology, biostatistics, and public health job opportunities, analyzes job requirements, and scores candidate fit using LLM-based profile matching
  • Automated personalized outreach workflows — generating tailored networking emails, cover letters, and faculty introductions based on each job description, research interests, publication history, and institutional priorities
  • Integrated Gmail API, Google Custom Search, and recruiter/faculty discovery pipelines to identify hiring contacts, track outreach status, and maintain a structured PostgreSQL database of applications, responses, and follow-up actions
  • Built as a self-directed tool to scale my own multi-track search across pharma RWE/HEOR, clinical epidemiology, and academic research roles
